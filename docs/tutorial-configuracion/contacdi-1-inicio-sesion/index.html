Ingresar al sistema

Inicio de sesión​

Para acceder al sistema de contabilidad se debe hacer por el primer formulario que nos muestra el sistema de contabilidad al abrirlo.

Ingresamos nuestro usuario de administrador para ener acceso a todas las funcionas del sistema.

Asegurece de ingresar el Usuario y contraseña validos.

Si ha olvidado su Usuario o contraseña debe contactar con el administrados del sistema para la recuperación de usuario o contraseña.

Crear nuevo usuario​

Para crear un nuevo usuario debe ingresar al menú configuraciones en la opción Usuarios opción Nuevo usuario

Al ingresar a esa opción del menú nos mostrara el formulario donde crearemos nuestro nuevo usuario y contraseña.

Pasos a seguir para crear un nuevo usuario desde el formulario: NUEVOS USUARIOS.

  • 1 - Presionamos el botón Nuevo.

  • 2 - Ingresamos el Usuario (Este es el primer parametro para iniciar sesión.)
  • 3 - Ingresamos el nombre de la persona a la cual se esta creando el usuario. (puede ser un nombre y un apellido).
  • 4 - Ingresamos la Contraseña (La contraseña puede ser mayúsculas o minúsculas)
  • 5 - En el siguiente cuadro: repetir contraseña debemos repetir la contraseña del paso número 4.
  • 6 - Elegimos el Tipo de Usuario que usara el sistema.
    • 6.1 - Usuario INVITADO solo tiene acceso a consultar los reportes. (Ideal para supervisar)
    • 6.2 - Usuario ADMINISTRADOR tiene acceso total al sistema. (Ideal para contador de CDI)

Así quedaria nuestro formulario terminado.

  • 7 - Presionamos el botón Guardar.

Usuario o contraseña

Si haz olvidado tu usuario o contraseña puedes contactar con el administrador del sistema sea por whatsapp personal o en el grupo de whatsapp del sistema contable, para programar una asistencia remota y proceder a la recuperación de dicha información.
